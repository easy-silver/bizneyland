# Biz & System - 상담관리 시스템

## 프로젝트 설명
고객 및 고객에 대한 상담 내용을 관리하는 시스템입니다.

## 사용한 기술
* Spring Boot 2.1.9
* JPA(Spring Data JPA)
* JUnit 4
* Thymeleaf
* AWS(EC2, RDS)
* OAuth2.0
